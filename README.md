# Лабораторная работа №5: Работа с базами данных (PostgreSQL)

## 📌 Описание

Проект представляет собой реализацию лабораторной работы по дисциплине «Скриптовые языки программирования». Разработана простая банковская система с использованием СУБД PostgreSQL и языка Python.

**Предметная область:** Банковская система  
**Технологии:** PostgreSQL, Python, SQLAlchemy, Docker

---

## 🏗️ Структура проекта

```
lab5/
├── .env.example          # Шаблон переменных окружения

├── config.py             # Конфигурация подключения к БД

```

## 🚀 Быстрый старт

### 1. Клонирование репозитория

```bash
git clone <URL_РЕПОЗИТОРИЯ>
cd lab5
```

### 2. Запуск через Docker

```bash
# Запуск всех сервисов (база данных + приложение)
docker compose up --build
```

### 3. Остановка

```bash
docker compose down
```

## ⚙️ Конфигурация

### Файл `.env`

| Переменная | Значение по умолчанию | Описание |
|------------|----------------------|----------|
| `DB_HOST` | `db` | Хост базы данных (внутри Docker) |
| `DB_PORT` | `5432` | Порт PostgreSQL |
| `DB_NAME` | `banking_db` | Имя базы данных |
| `DB_USER` | `postgres` | Пользователь БД |
| `DB_PASSWORD` | `postgres` | Пароль пользователя БД |

## 📊 Схема базы данных

### Таблицы

| Таблица | Описание | Связи |
|---------|----------|-------|
| `clients` | Клиенты банка | — |
| `accounts` | Счета клиентов | 1:N с clients |
| `transactions` | Транзакции по счетам | 1:N с accounts |

### ER-диаграмма

```
┌─────────────┐       ┌─────────────┐       ┌────────────────┐
│   clients   │       │   accounts  │       │  transactions  │
├─────────────┤       ├─────────────┤       ├────────────────┤
│ id          │───┐   │ id          │───┐   │ id             │
│ username    │   │   │ client_id   │◄──┘   │ account_id     │◄──┐
│ email       │   └──►│ account_num │   │   │ amount         │   │
│ created_at  │       │ balance     │   └──►│ type           │   │
└─────────────┘       └─────────────┘       │ description    │   │
                                            └────────────────┘
```

## 📝 Выполняемые операции

### CRUD через ORM

- **Create** — создание клиента и счёта
- **Read** — фильтрация клиентов по email
- **Update** — изменение баланса счёта
- **Delete** — удаление клиента

### Raw SQL

- Вставка транзакции через прямой SQL-запрос
- Подсчёт количества записей в таблице

### Транзакции

- Атомарное создание клиента и счёта
- Откат изменений при ошибке (`rollback`)

## 🔌 Подключение к базе данных

### Из хост-машины (pgAdmin, psql)

```bash
# Через psql
psql -h localhost -p 5433 -U postgres -d banking_db

# Пароль: postgres
```

### Из контейнера

```bash
docker compose exec db psql -U postgres -d banking_db
```
